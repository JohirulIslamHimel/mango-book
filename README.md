# 📚 MangoBook - Modern Library Management System

**MangoBook** is a modern web application where users can browse, search, and discover books from various categories. Built with **Next.js** and **Better-Auth**, it provides a seamless experience for readers to manage their profiles and explore trending titles.

## 🚀 Live Demo

## 🎯 Project Purpose

The goal of this project is to create a user-friendly platform for book enthusiasts. It focuses on clean UI/UX, secure authentication, and efficient data handling, making it a great example of a modern full-stack application.

## ✨ Key Features

- **User Authentication:** Secure Sign In and Sign Up using **Better-Auth** (supports Email/Password and Google Login).
- **Dynamic Book Catalog:** A complete list of books with real-time search and category filtering.
- **Interactive Slider:** Features a "Trending Books" section using **Swiper.js** for a smooth browsing experience.
- **Protected Routes:** Using **Middleware** to ensure only logged-in users can access profile and book detail pages.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using **Tailwind CSS** and **DaisyUI**.
- **Real-time Notifications:** Integrated **React Hot Toast** for success and error messages during user actions.

## 🛠️ Technologies Used

- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS & DaisyUI
- **Authentication:** Better-Auth
- **Database:** MongoDB
- **Frontend Tools:** Swiper.js, React Hot Toast
